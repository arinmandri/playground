## 목적

특정 목적의 서비스라기보단 그냥 개발자로서 내가 할 줄 아는 것을 써보거나 시험한다.


## 서비스 구조

DB, 사용자 프론트&백
XXX 관리자 페이지는 일단 없음.

현재 서버의 물리적 실체는 본가에 둔 컴퓨터본체인데; 24시간 돌아가니 신경쓰인다는 아빠의 의견을 반영하여 쓸 때만 WOL을 이용하여 켜고 평소에는 꺼둔다.



## 기술스택

* 프론트엔드: TypeScript, Vue 3
* 백엔드: Maven, Spring Boot 3, Hibernate
* DB: PostgreSQL
* 그외: Nginx, PM2
